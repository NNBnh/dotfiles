## 🚀 Setup

Install [Arch](https://archlinux.org) then run this command on the terminal:

```sh
BINDIR="$HOME/.local/bin" sh -c "$(curl -fsLS chezmoi.io/get)" -- init --apply NNBnh
```

That's all fork!
